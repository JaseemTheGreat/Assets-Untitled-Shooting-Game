# Assets-Untitled-Shooting-Game
These are Assets for Untitled Shooting Game

In all the files, they will use arrays `[]` in order to function.

In `/Costumes/Enemies/`, all enemies have a datauri corresponding to their image, so when the game imports them, they will be recognised.
They are formatted like this: `["defaultcostume", "stuncostume", "punch1costume", "punch2costume"]` and the rest of the objects will correspond to their abilities

In `/Data/Enemies/` The arrays are formatted like this: `["name", "hp", "speed", "turnrate", "defense", "boss-status"]`
